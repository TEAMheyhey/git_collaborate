# 🍜 주제 1: 인천의 역사와 함께하는 노포 맛집

인천 하면 떠오르는 상징적인 맛집을 소개합니다.

## 1. 신승반점 (인천 차이나타운)
- **특징**: 짜장면의 원조 '공화춘'의 후손이 운영하는 곳
- **추천 메뉴**: 유니짜장, 찹쌀탕수육
- **선정 이유**: 진한 춘장의 맛과 쫄깃한 면발이 일품이며 역사적 의미가 큼

## 2. 화평동 할머니 냉면 (세숫대야 냉면)
- **특징**: 엄청난 양과 저렴한 가격으로 유명한 냉면 거리의 원조
- **추천 메뉴**: 물냉면, 비빔냉면
- **선정 이유**: 가성비가 매우 뛰어나고 인천 시민들의 추억이 담긴 장소
